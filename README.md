# Gosaic

## Usage

Build with docker:

```
位 make build
```

Use the built application:

```
位 ./gosaic [...arguments]
```

Examples:

```
位 ./gosaic mosaic aspect /images/macro/final/b30_original.jpg -s 40 -w 9000 -n b30_40_9000

位 ./gosaic mosaic aspect /images/atesz/02.jpg -s 40 -w 3000 -n atesz
```
